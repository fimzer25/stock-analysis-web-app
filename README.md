# Stock Analysis Web Application

A client-side web application for analyzing stock market trends using technical indicators and basic prediction models.

## Features
- Interactive stock charts
- Technical indicators (RSI, MACD)
- Basic price prediction using linear regression
- Stock search functionality
- Buy/sell simulation with balance tracking
- Responsive UI (mobile & desktop)

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download or clone the repository
2. Open index.html in your browser

## Purpose
This project was developed to explore stock data visualization, technical analysis, and basic predictive techniques in a web-based environment.
